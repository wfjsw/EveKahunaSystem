# 什么是 Kahuna System

KahunaSystem是一款以流程和方法论为核心开发00工业辅助平台，最终目的是为熟练度不足或是希望节省精力的玩家提供一整套易理解、可操作的执行方案，按部就班即可生产出产品，降低门槛。

该方法论在紫竹梅公司内部已经实现帮助3位完全没有接触过工业的玩家生产出一批伊什塔，虽然他们搓完后仍然无法回答T2工业怎么玩。

由于本开发者并不是体量很大的制造商，方案甚至整体架构设计可能在超大型规模上考虑不周，欢迎提出宝贵意见。

# 如何实现？
本平台致力于通过简单易懂易操作的方式同时回答以下几个重要问题：

- 我该搓什么？
    - 自选利润监控与市场分析【大饼】
- 搓出来赚钱吗？
    - 提供详略得当的成本报表，心里有底。
- 我该怎么搓？
    - 通过监控库存和流水线，实现输出可执行的工作流建议和物流建议，照着搬运和进行工作即可。
